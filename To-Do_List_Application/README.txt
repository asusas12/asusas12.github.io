Our To-Do List Application is a simple web-based CRUD system that allows users to manage their daily tasks efficiently. It is built using HTML, CSS, and JavaScript for the frontend, and PHP with MySQL for the backend and database.

The system connects to a MySQL database named todo_db, which contains a tasks table. This table stores each task’s ID, title, description, status (pending or completed), and the date it was created. The db.php file handles the database connection, while api.php manages all CRUD operations such as adding, viewing, updating, and deleting tasks through different HTTP request methods.


On the frontend, users can add new tasks, edit existing ones, mark tasks as completed, delete tasks, and filter them by status (All, Pending, Completed). The interface is clean and user-friendly, with task cards, buttons, and toast notifications for feedback. Overall, the application demonstrates how a full-stack CRUD system works by connecting the user interface to the database in a simple and organized way.
